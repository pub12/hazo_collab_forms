# Profile Picture Library

This directory contains library profile pictures organized by category.

## Directory Structure

- Each subdirectory represents a category (e.g., `Cars/`, `Young Cartoons/`)
- Place image files (JPG, PNG) in the appropriate category directory
- Supported formats: JPG, PNG
- Recommended size: 200x200px

## Adding New Categories

1. Create a new subdirectory with the category name
2. Add image files to the directory
3. Images will automatically appear in the library tab when the category is selected

## Adding Images

1. Place image files in the appropriate category directory
2. Use descriptive filenames
3. Ensure images are properly formatted (JPG or PNG)
